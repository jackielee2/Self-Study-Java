12. 조건문 ( if, switch )..
==========================================================

	---------
	1)	if문..
			[ 참고 ] :	[ If_Test_1.java ]
						[ If_Test_2.java ]
	---------

		-	형식..

			1)	if( 조건식 )
				{
					실행문 - true..
				}

				-	[ 실행문 - true ]의 실행 문장이
					한 줄이면 중괄호 생략 가능..

				-	중첩 가능..

					if( 조건식 )
					{
						if( 조건식 )
						{
							실행문 - true..
						}
					}




			2)	if( 조건식 )
					
					실행문 - true..

				else

					실행문 - false..




			3)	if( 조건식 - 1 )
					
					실행문 - true..

				else if( 조건식 - 2 )
					
					실행문 - true..

				else if( 조건식 - 3 )
					
					실행문 - true..

				else
					
					실행문 - false



	-------------
	2)	switch문..
			[ 참고 ] :	[ Switch_Test_1.java ]
						[ Switch_Test_2.java ]
						[ Switch_Test_3.java ]						
	-------------

		-	형식..

			switch( 변수 )
			{
				case 값1 :
					실행문 - 1;
					break;

				case 값2 :
					실행문 - 2;
					break;

				case 값3 :
					실행문 - 3;
					break;

					....

				default :
					실행문 - ...;
					break;
			
			}

			-	[ if ]는 경우의 수가 많아질수록
				[ else - if ]를 반복적으로
				추가해야 하므로 코드가 복잡해짐..

				ㄴ	[ switch ]는 [ if-else if ]에
					비하여 코드가 간결함..


			-	[ case 문 ] 끝에 [ break ]로
				마무리 하지 않으면
				다음 [ break ]를 만날때까지
				모든 실행문을 처리..


			-	[ default 문 ] 실행..

				ㄴ	[ case ]에 해당하는
					값이 없다면 실행..

				ㄴ	[ if - else if ]의
					마지막 [ else ]과 유사..
